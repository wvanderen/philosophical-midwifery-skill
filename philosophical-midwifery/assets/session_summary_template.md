## Filled-in example — "recognition determines worth"

## Session Summary *(example)*

**Presenting disturbance:**
The user feels resentful when their work goes unrecognized.

**Concrete example:**
A project the user led shipped successfully, but no one acknowledged their contribution, and the user felt the work "didn't happen."

**Candidate pathologos:**
*Conditional worth* — offered as one hypothesis that fit the example (not asserted as the user's belief).

**Examined proposition:**
"The value of work depends on recognition." *(user-confirmed, normalized)*

**Definitions:**
- *recognition*: someone noticing or acknowledging the work
- *worth*: the value of the work itself, independent of its visibility

**Implications tested:**
- If recognition determines worth, then unrecognized work has no value.
- If recognition determines worth, then any valuable work that is never seen is worthless.

**Counterexamples found:**
- The user agreed a colleague's excellent unnoticed work would still be good.

**Contradiction or weakness:**
The user holds that unnoticed work can still be good, yet felt their own unnoticed work had no value — the two commitments cannot both hold as stated.

**Revised proposition:**
"Recognition affects visibility and opportunity, but does not determine intrinsic worth." *(user-endorsed restatement)*

**Unresolved tension:**
The user still equates professional invisibility with personal diminishment, even after accepting the revision conceptually.

**Next inquiry target:**
Examine the proposition "Being overlooked means being diminished" — the thread that did not resolve.
